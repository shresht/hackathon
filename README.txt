Phillips Hackabout 2017 Solution
Team leader - Sushruth
================================================
Instructions:

1. Install Anaconda3 from https://www.continuum.io/downloads

2. Create virtual environment with all requirements
---------------------------------------------------
Open a terminal in the same directory ("hackathon"). Run the following command:
conda env create -f environment.yml

3. Download the GloVE word embeddings from https://drive.google.com/uc?export=download&id=0B30g1WfHiiY-MmN2dVVkdnV1S2M

4. Place the glove.6B.100d.txt file in the same directory ("hackathon")

5. Open a command terminal in this directory ("hackathon") and run:
------------------------------------------------------------------
activate tensorflow
jupyter-notebook Hackathon.ipynb

6. Wait for the notbook to load up in a browser

7. Press Shift+Enter to run each cell

8. The last cell writes the results of the test set to results.txt 

Feel free to contact tornadoalert@gmail.com for any queries.
If anything does not work, the result of the notbook is available at https://github.com/tornadoalert/hackathon/blob/master/Hackathon.ipynb
